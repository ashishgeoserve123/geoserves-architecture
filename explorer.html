Black Box Expansions

BB-1 — Domain APIs (Service Region)

  • User API — JWT auth, users/roles/policies, SSO, Graph
  • Masters API — ports, fuels, vessel classes, CP terms
  • Voyages API — fleet analytics, MRV, AER, FuelEU, Hangfire jobs
  • Reports API — Excel reports via ClosedXML templates
  • Library API — API-key integration, MediatR, GeoOne proxy

BB-2 — Background Workers (Worker Region)

  • Report Engine — QuestPDF render, blob upload, Hangfire retry
  • Charterparty Jobs — TCV2 (7 workers) + TCV2LNG (5 workers)
  • Digital Twin — DTAPI + DTScheduler + Python ML models
  • Data Verification — ABP layered, DNV, Verifavia integration

BB-3 — Infrastructure Layer

  • AWS Secrets Manager — config / secrets for all services
  • AWS CloudWatch Logs — centralised logging
  • Prometheus — metrics scraping (/metrics endpoints)
  • Hangfire — recurring job scheduling for Voyages + CP Jobs
  • ABP Services — framework services for DataVerification
  • Application Insights — APM for Voyages API

BB-4 — External Integrations

  • Microsoft Graph API — email notifications, user data
  • GeoOne SSO — single sign-on integration
  • Google reCAPTCHA — bot protection on auth flows
  • Weather APIs / DTN — weather data for exclusion calc
  • Distance API — voyage distance calculations
  • Azure Blob / S3 — PDF/Excel storage, ML models
  • DNV + Verifavia — external data verifiers
  • Gotenberg — PDF rendering for Digital Twin
  • Python Runtime — ML model execution (DTScheduler)
